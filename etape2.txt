1/ Créer un fichier avec la commande "touch". Exemple : touch etape2.txt

2/ Faire un git status puis un git add. Exemple : git add etape2.txt

3/ Faire un commit. Exemple : git commit -m "Votre texte ici" 
   NB : si il existe déjà un commit on utilise le raccourci : git commit -am "Votre texte ici"

4/ Envoyer le commit local vers github avec la commande push. Exemple : git push origin master
