Pour le projet de l'activité GIT :
Etape 1/ Créer le répertoire activite_git 
Etape 2/ Créer et ajouter le fichier README.txt qui présente le projet
Etape 3/Au moins 2 autres fichiers de votre choix (ils peuvent contenir des scripts de code, du texte sur un projet personnel...);
Etape 4/Au moins 4 commits distincts montrant l’ajout puis la modification de certains fichiers.
Etape 5/ Envoyer le repository Git vers le GitHub !
