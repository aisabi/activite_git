Ce repository décrit de manière succinte les étapes pour utiliser Git.
Il ne saurait remplacer un cours. Celui d'OC est très bien pour acquérir les bases.
Deux étapes pour décrire les fondamentaux de l'utilisation de Git.

Rappel des éléments demandés pour l'activité git.
Pour le projet de l'activité GIT :
1/ Créer le répertoire activite_git 
2/ Créer et ajouter le fichier README.txt qui présente le projet
3/Au moins 2 autres fichiers de votre choix (ils peuvent contenir des scripts de code, du texte sur un projet personnel...);
4/Au moins 4 commits distincts montrant l’ajout puis la modification de certains fichiers.
5/ Envoyer le repository Git vers le GitHub !
